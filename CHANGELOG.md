# 更新日志

## v1.2.0 (2026-05-29)

### ✨ 新增
- 欧洲节点 (EU) 的关键词识别与自动分组
- 学术专属分流策略 (Sci-Hub、Google Scholar 等)，国内学术库保持直连
- 全局拦截 UDP 443 端口 (QUIC) 配置（默认注释，按需开启），解决潜在 DNS 泄漏问题。
- 添加 **Sniffer 嗅探配置** ，会试图解析真实域名，提高分流命中率。

### 🔧 优化
- 完善局域网 (LAN) 及保留 IP 的直连规则，防止内网流量外泄
- 调整 URL-Test 测速地址为 `cp.cloudflare.com/generate_204`，加入 `lazy` 模式，降低空闲负载
- 调整并精简分流规则顺序，提升内核匹配性能
- 优化规则集 (rule-providers) 的下载代理为“自动选择”，解决首次启动时因手动节点未选择导致规则下载卡死的死锁问题。


### 📝 修复
- 修正 `game-download` 规则集路径（移除无效的 `@cn` 后缀）
- 修正 `safe` 函数对空区域组的处理，避免 `REJECT` 导致故障转移提前终止
- 修复游戏代理组的下载流量未直连，图片加载未走代理的问题
- 修复 `其他节点` 为空时的 fallback 逻辑，改为 `DIRECT`，避免故障转移链中断
- 修复因纯英文字符（如 US、DE）导致的节点名称误分类问题（引入正则边界断言解决）。
